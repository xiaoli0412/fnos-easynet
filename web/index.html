fnos-easynet 系统配置与说明

关于

fnos-easynet 是专为飞牛OS (fnOS) 设计的 Docker 化网络代理解决方案。

核心基于 mihomo (Clash Meta)，提供高性能的多协议代理支持。

代理模式：通过 HTTP (7890) / SOCKS5 (7891) 端口提供应用层代理，需在应用中手动配置代理地址。适合单个应用的代理需求。

TUN 模式：创建虚拟网卡接管系统全部网络流量，实现透明代理。无需逐个应用配置，全局生效。需要 NET_ADMIN 权限。

飞牛OS适配：本方案针对飞牛OS (基于 Debian) 优化，支持通过 Docker Compose 一键部署，完美兼容飞牛OS的Docker管理界面。

飞牛OS 部署指南

1. 准备配置文件

在飞牛OS文件管理中创建目录 /vol1/docker/fnos-easynet，将 docker-compose.yml 和 config.yaml 放入该目录。

2. 编辑订阅配置

编辑 config/config.yaml，将 proxy-providers 中的 URL 替换为你的订阅链接。

3. 启动容器

通过 SSH 进入目录执行 docker compose up -d，或在飞牛OS Docker管理界面导入 Compose 配置。

4. 访问面板

浏览器打开 http://飞牛IP:9090 即可进入管理面板。
